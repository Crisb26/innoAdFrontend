# InnoAd Frontend

Sistema de Gestion de Publicidad Digital - Interfaz de Usuario

## Descripcion

Frontend desarrollado en Angular para la plataforma InnoAd.

## Tecnologias

- Angular 19
- TypeScript 5.7
- NgRx para gestion de estado
- NgBootstrap
- Chart.js
- Leaflet
- RxJS

## Requisitos

- Node.js 20.18 LTS o superior
- npm 10.8.2 o superior
- Angular CLI 19

## Instalacion

1. Instalar dependencias: npm install
2. Ejecutar en desarrollo: ng serve

Aplicacion disponible en http://localhost:4200

## Scripts

- npm start - Servidor de desarrollo
- npm run build - Construccion para produccion
- npm test - Pruebas unitarias
- npm run lint - Analisis de codigo

## Funcionalidades

- Autenticacion de usuarios
- Dashboard con metricas
- Gestion de pantallas Raspberry Pi
- Gestion de contenidos multimedia
- Perfil de usuario

## Construccion para Produccion

npm run build:prod

Archivos generados en dist/innoad-frontend
